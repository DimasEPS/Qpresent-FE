# Qpresent FE

Antarmuka Qpresent menyiapkan pengalaman absensi berbasis QR dan verifikasi GPS untuk mahasiswa dan dosen. 

## Teknologi
- React 19 + Vite untuk pengembangan front-end modern.
- React Router 7 untuk navigasi multi-halaman.
- Tailwind CSS 4 beserta utilitas shadcn/ui untuk styling konsisten.
- Lucide React sebagai paket ikon.

## Menjalankan Proyek
1. Install dependensi
   ```bash
   npm install
   ```
2. Mode pengembangan
   ```bash
   npm run dev
   ```
3. Build produksi
   ```bash
   npm run build
   ```
4. Preview hasil build
   ```bash
   npm run preview
   ```
